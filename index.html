image-size

Node module for detecting image dimensions

View the Project on GitHub netroy/image-size

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Instalation

npm install image-size

Usage

var sizeOf = require('image-size');
var dimesions = sizeOf('images/funny-cats.png');
console.log(dimesions.width, dimesions.height);

Supported formats

  • BMP
  • GIF
  • JPEG
  • PNG
  • PSD
Upcoming
  • TIFF
  • WebP
